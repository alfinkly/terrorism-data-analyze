# Product Overview

This is a terrorism data analysis project focused on analyzing the Global Terrorism Database (GTD) with particular emphasis on Central Asia and Kazakhstan. The project provides statistical analysis, visualizations, and comparative studies of terrorist incidents across different regions, countries, and time periods.

## Key Features

- Regional terrorism analysis with focus on Central Asia
- Country-specific deep dives (especially Kazakhstan)
- Per-capita attack rate calculations
- Weapon type and terrorist group analysis
- Time-series trend analysis
- Comparative ranking systems

## Data Sources

- Primary dataset: `gtd.xlsx` (Global Terrorism Database)
- Development dataset: `gtd-mini.xlsx` (subset for faster development)
- External population data via `countryinfo` library for per-capita analysis

## Output Formats

- Statistical reports printed to console
- PNG visualizations saved to project root
- Matplotlib/Seaborn charts for data presentation
